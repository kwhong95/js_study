# 연산자, 기타
## 1. Trailing commas
- **Object** 끝에 콤마 사용
  + } 앞에 콤마 사용 가능
- **배열** 끝에 콤마 사용
  + ] 앞에 콤마 사용 가능
```js
const obj = {
  book: 100,
  point: 200,
};

const list = [100, 200,];
```

## 2. 거듭 제곱
- **거듭 제곱**
```js
console.log(2 ** 3);
console.log(3 ** 2);

console.log(2 ** 3 ** 2);
console.log(2 ** (3 ** 2));
console.log((2 ** 3) ** 2);

[실행 결과]
8
9
512
512
64
```
1. `2 ** 3 ** 2`  
2의 3승의 2승이 아니라
2. 먼저 3의 2승을 구하며 값이 9  
2의 9승 이므로 512

- **좌결 합성**
  + 왼쪽에서 오른쪽으로 계산
  + `1 + 2 + 3은 (1 + 2) + 3`으로 계산
- **우결 합성**
  + 오른쪽에서 왼쪽으로 계산
  + `A ** B ** C`에서 `A ** (B ** C)`로 계산
